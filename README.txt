# RAG-Teste: Assistente Local com Streamlit

## Como rodar

1. (Opcional) Crie um ambiente virtual:
   python -m venv venv

2. Ative o ambiente:
   - Windows: .\venv\Scripts\activate
   - Linux/Mac: source venv/bin/activate

3. Instale as dependências:
   pip install -r requirements.txt

4. Rode o assistente:
   streamlit run app.py

O assistente irá abrir no navegador. Faça perguntas em inglês sobre os experimentos.
